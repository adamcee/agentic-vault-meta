# AI Writing Discipline

The Agentic Vault treats project documentation as a **high-leverage state machine for multi-agent LLM parsing**. 

To prevent context rot, token bloat, and degraded attention mechanisms in the LLM, we adhere to strict writing protocols across all vault documents.

## BLUF (Bottom Line Up Front)
Every document and major section must start with the most critical information. Do not use introductions.

## The Paragraph Ban
Limit standard paragraphs to a maximum of two sentences. If a thought requires more explanation, synthesize it into a bulleted list with bolded lead-in terms.

## AI Silence
Do not use conversational filler or meta-commentary like "This document will explain..." or "As previously discussed." State the facts directly.

## Reification
Do not name abstract observations. If two things differ, describe the difference. Do not capitalize the difference, give it a section header, or declare it a principle.
